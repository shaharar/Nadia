About Nadia


Nadia Comaneci is a Romanian retired gymnast and a five-time Olympic gold medalist, all in individual events.

She is the first gymnast to be awarded a perfect score of 10.0 at the Olympic Games (1976 Summer Olympics in Montreal).

During her career, Comaneci won nine Olympic medals and four World Artistic Gymnastics Championship medals.

Q&A

Three main differences between Web 1, Web 2 and Web 3:

First, Web 1 is considered "Read-only web" - the content was based on the producer of a site,
Web 2 is considered "Read-and-Write web" - so that the content creativity came from both users and producers,
and Web 3 targets on linked data sets.

Second, as for the main feature - Web 1 is called "The Hypertext Web", Web 2 is said to be "The Community Web",
and Web 3 is considered "The Semantic Web".

Third, while Web 1 was used for information sharing (Corporate websites), Web 2 connected people (Social networks),
and Web 3 is said to be connecting knowledge (Wisdom of the crowd).


Examples of the differences that mentioned above:

* Web 2 brought us "Wikipedia" - that allowed users to edit and enrich pages - unlike Web 1,
where these options were only open to the site owner.

* Queries - in Web 2 era we Googled it (general search engine), Web 3 introduced professional,
expert-knowledge based search engines (such as treato).

* Simple, minimal pages in Web 1 - were replaced with friends-connecting social media platforms like Facebook and Twitter.
